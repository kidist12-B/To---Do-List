# 📝 To-Do List App

A simple and clean To-Do List application built using **HTML** and **CSS**. This project was created as a beginner web development project to practice page structure, layouts, forms, Flexbox, and UI design.

## 📸 Overview

The application contains:

* A header section with the application title and description
* An input field for adding tasks
* Task statistics cards
* A task list with checkboxes
* A clean and organized user interface

## 🚀 Features

* Add Task input area
* Statistics dashboard

  * Active Tasks
  * Completed Tasks
  * Total Tasks
* Task cards with checkboxes
* Responsive layout using Flexbox
* Simple and user-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
